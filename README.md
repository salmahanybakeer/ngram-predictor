# N-Gram Next-Word Predictor

A next-word prediction system built from scratch using an n-gram
language model, trained on Sherlock Holmes novels.

## Requirements
- Python 3.11
- Install libraries via `requirements.txt`

## Setup
1. Clone this repo
2. Create and activate an Anaconda environment:
   `conda create -n ngram-predictor python=3.11`
   `conda activate ngram-predictor`
3. Install dependencies:
   `pip install -r requirements.txt`
4. Fill in `config/.env` with the required paths (see file for format)
5. Download the raw `.txt` files (see Dataset section) into `data/raw/train/`
